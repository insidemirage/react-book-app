# React Book App
This app was created as test, and doesn't processes some specific cases.

## Development
First of all u need to create .env file in root directory:   
```
REACT_APP_API_KEY = YOUR_KEY
```
Install packages:   
```bash
yarn install
```   
Run react:
```bash
yarn start
```
Build project:
```bash
yarn run build
```
